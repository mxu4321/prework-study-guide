HTML

  • The body element represents the visible content shown to the user.
  • "h1"should only appear once on each page.
  • Alt text can also improve search engine rankings for your website.
  • An HTML developer is also responsible for apositive user experience.
  • Elements that are nested inside other elements are called child elements.

  • A tag refers only to what appears inside the angle brackets.

Git

  • Git is a version control system.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • A branch is a copy or version of the code that can be worked on independently.
  • It's important to frequently sync the local code with the remote version in GitHub.
  • A pull request, or PR, is used to merge a comparison branch into the base branch.
  • Debugging is one of the most important parts of the development process.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • console.log is a method to output a message to the we console.
  • An array is a single variable that is used to hold a group of data.
  • Functions are reusable blocks of code that contain instructions to perform a specific task.
  • ''for'' loop: 1.keyword "for"; 2.three-part conditional statement; 3.code block.
